01 // PURPOSE & BACKGROUND

Why did we build this?

We needed a self hosted translator for translation of incoming signals (messages, filenames etc...) from OSINT Tools like TeleGather.

02 // ARCHITECTURE

How It Works Under The Hood

Step 01

Tokenization & Segmentation

The .NET backend parses text using the Catalyst NLP engine to split sentences and words while preserving whitespace.

Step 02

Dictionary Bypass

Each token is checked against a 73k-word English dictionary. Verified English segments bypass translation completely.

Step 03

Smart Batching

Non-English segments are grouped by script and translated via LibreTranslate or TranslateGemma (Ollama).
