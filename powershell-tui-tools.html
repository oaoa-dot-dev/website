Products / PowerShell Console TUI Toolkit

PowerShell
TUI Tools.

A premium suite of modern console Text User Interface (TUI) utilities replicating classic Windows administration tools. Built entirely in PowerShell with raw ANSI escape sequences, these utilities provide rich visual aesthetics, custom color palettes, smooth scrollable lists, dialog confirmations, and responsive layouts.

View on GitHub PowerShell Gallery Profile
Architectural Cleanliness

Zero Dependencies

Built entirely using native PowerShell scripting and standard ANSI escape sequences. No external DLLs, frameworks, or wrappers required—ideal for headless server environments.

DevOps Integration

DevOps Build Pipeline

Includes a compilation pipeline script (build.ps1) that automatically bundles driver scripts and library components into single, standalone TUI execution scripts.

Deployment

PSGallery Registered

All scripts are published as standalone scripts in the public PowerShell Gallery, making deployment as simple as running a single Install-Script command on target servers.

Quick Start

Install the Full Suite

Run the following command in an administrative PowerShell terminal to download and install all 7 TUI utilities to your system path.

Install-Script -Name EventViewerTUI, RecMonTUI, DeviceManagerTUI, DevicesPrintersTUI, ProgramsFeaturesTUI, ServicesTUI, MSInfo32TUI -Scope CurrentUser -Force

Note: Standard PowerShell gallery security prompts will appear unless -Force is passed. Once installed, simply type any tool name (e.g. EventViewerTUI) to launch.

The 7 Administrative Utilities

01 // Event Viewer TUI

EventViewerTUI

Browse and query local Windows Event Logs inside the terminal. Replicates the core tree and list workflow of eventvwr.msc.

  • Dynamic tree grouping (Windows Logs vs Application and Services Logs).
  • Advanced Filter Window (Ctrl+F or Ctrl+1) supporting severity toggles.
  • Full search text and Event ID filtering on remote sessions.
Install-Script -Name EventViewerTUI

02 // Resource Monitor TUI

RecMonTUI

03 // Device Manager TUI

DeviceManagerTUI

A full-featured terminal manager for connected system hardware. Integrates the operational capabilities of devmgmt.msc.

  • View devices grouped by type or by connected bus hierarchy.
  • Actions: Scan for changes, Update driver, Enable/Disable, and Uninstall devices.
  • Disabled devices are colored red, hidden devices are shown in gray.

04 // Devices & Printers TUI

DevicesPrintersTUI

A centralized TUI control panel for printer queues, printer ports, and active hardware devices. Replicates the classic Windows Control Panel applet.

  • Scans and manages local and user-session mapped printers.
  • Parses HKEY_USERS SID registry hives to view profiles across other logged-in users.
  • Actions: Set defaults, add TCP/IP ports, delete device queues.

05 // Programs & Features TUI

  • Deep registry auditing scanning both 32-bit and 64-bit application paths.

06 // Services Manager TUI

ServicesTUI

A full Explorer-style console TUI for inspecting and operating Windows background services. Replaces services.msc.

  • Split-pane layout: category grouping trees on the left, service status grids on the right.
  • Operations: Start, Stop, Pause, Restart, and change startup state (Auto, Manual, Disabled).
  • Built-in details panel view (I key) displaying WMI properties.

07 // System Information TUI

MSInfo32TUI

A hierarchical category tree interface auditing hardware and system configurations. The CLI equivalent of msinfo32.exe.

  • Dynamic WMI/CIM queries loading categories on-demand to limit memory footprint.
  • Comprehensive tree coverage: Hardware Resources (IRQs, DMA), Components, and Software.
  • Bottom status bar summarizing active category data.
