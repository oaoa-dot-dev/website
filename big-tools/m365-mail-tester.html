Test delegated mail send via Microsoft 365. Sign in with your M365 account, compose a message, and inspect the full Graph API response.

How This Works Clicking "Sign in with Microsoft" opens a small popup window that handles the OAuth 2.0 redirect flow with Microsoft. Once you authorise, the popup closes automatically and this page unlocks. Mail is sent via the Microsoft Graph API (/v1.0/me/sendMail) using delegated Mail.Send permissions. Your credentials are never stored — the API holds only a short-lived JWT session token.
