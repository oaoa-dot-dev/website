What You Need
  1. SMTP AUTH enabled for the sending account in Exchange Online Admin (Exchange Admin → Mailboxes → [account] → Mail flow settings → SMTP AUTH).
  2. An Exchange Online mail flow connector configured to accept SMTP relay from this tool's outbound IP: fetching...
  3. Your tenant OAuth URL — typically login.microsoftonline.com/{tenant-id}.

The API server connects to smtp.office365.com:587 and authenticates via AUTH XOAUTH2. No Entra app registration is required — authentication uses Microsoft's own first-party OAuth endpoint for your tenant.
